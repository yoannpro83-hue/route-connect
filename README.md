# Route Connect

Site vitrine de **Route Connect** — création de sites web sur-mesure.
« Des sites qui prennent racine. »

La page d'accueil propose trois directions artistiques au choix :

- **Terminal** (`terminal/`) — near-black, monospace, vert électrique, esthétique routing/code.
- **Émeraude organique** (`emeraude/`) — forêt sombre, émeraude & laiton, système racinaire animé en fond.
- **Clair éditorial** (`edito/`) — ivoire, grosse typo serif, accent cobalt.

Sites statiques (HTML/CSS/JS), responsive, sans dépendance.

En ligne : https://yoannpro83-hue.github.io/route-connect/
